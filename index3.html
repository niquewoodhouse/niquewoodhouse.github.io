Product design

WIREWAX

The first ever free to use HTML5 video editor that can make anything in any video interactive. As Creative Director since 2013, I've had the privilege of designing a product used by some of the world's most renowned brands.

How it works in 3

  • 1 Upload Drag n drop videos or paste URLs from popular sites like YouTube.

  • 2 Tag Make anything in the video interactive by just drawing around it.

  • 3 Add Use a simple drag & drop interface to design what appears when viewers interact.

UI patterns

I placed videos into a card interface to give both new users an inbuilt frame for understanding the product, & the more experienced quick access to the tools they regularly need.

Using cards helps create a pattern that is both scalable to any device/screen & allows ui metaphors to be expressed visually.

  • Revision history A card system can be used to represent time & go through past content edits.

    • -

  • Drag to combine The visual of video cards combining to create a project helps explain how projects work.

  • Multi-select tools Contextual tools when a user selects multiple videos for power moves like delete all.

  • Explaining Cards are useful to help explain features in bitesize chunks.

    • -

  • Wearables Cards help create a scalable system that can be future-proof.

Identity design

WIREWAX

Alongside designing the product, I completely redesigned & helped expand the brand as offices opened in NYC, San Fransisco & LA.

Logo

A week long process involving 100+ ideas led to the final, approved logo. It was important to give a nod to the tech, but more so to dress it in a friendly minimal identity that could play well with various potential user brands.

Working closely with the brilliant CEO who wanted as many options as possible, we went from 100s of sketches to 10s of prototypes. Tweaking type to infer a pause icon within the name, wire, dynamic shapes, nostalgic illustrations. We went round the houses.
